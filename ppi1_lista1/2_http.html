Protocolo HTTP

O HTTP (HyperText Transfer Protocol) é um protocolo utilizado na Web para a comunicação entre o navegador (cliente) e o servidor. Ele funciona através de requisições (requests) feitas pelo cliente e respostas (responses) enviadas pelo servidor.

Exemplo de requisição HTTP

GET /index.html HTTP/1.1
Host: www.exemplo.com
User-Agent: Mozilla/5.0
Accept: text/html

Exemplo de resposta HTTP

HTTP/1.1 200 OK
Content-Type: text/html
Content-Length: 1024

Detalhes dos campos

Request

  • GET: método da requisição
  • /index.html: recurso solicitado
  • HTTP/1.1: versão do protocolo
  • Host: endereço do servidor
  • User-Agent: navegador utilizado
  • Accept: tipo de conteúdo aceito

Response

  • 200 OK: status da resposta
  • Content-Type: tipo do conteúdo
  • Content-Length: tamanho da resposta
