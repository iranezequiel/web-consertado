# Projeto PPI
Esse projeto foi feito para a aula de Programação para Internet.
Aqui eu fiz algumas atividades usando HTML e CSS para aprender o básico de como criar páginas web.

## O que tem no projeto
- Página básica em HTML
- Explicação sobre HTTP
- Tabela com estados
- Formulário de contato
- Estilo com CSS

## Observação
Foi uma atividade para praticar e entender melhor como funciona a criação de sites.
