Diferença de Internet e Web

A Internet é a infraestrutura física global(cabos, servidores, roteadores) que conecta computadores mundialmente. A Web (World Wide Web) é o sistema de páginas, documentos e hipermídia que trafega por essa rede, acessado via navegadores. Resumindo: a Internet é a rede física, enquanto a Web é o conteúdo que usamos nela.

História da Internet e da Web

A Internet surgiu na década de 1960 com o projeto ARPANET.

Em 1989, Tim Berners-Lee criou a Web, permitindo acessar páginas por navegadores.

A Web utiliza o protocolo HTTP para troca de dados.

Com o tempo, surgiram versões como HTTP2 e HTTP3.

Principais Marcos

  • 1969: ARPANET
  • 1983: TCP/IP
  • 1989: Web
  • 1990: Primeiro navegador

Evolução

  2. Uso acadêmico
